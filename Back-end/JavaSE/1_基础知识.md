# 一.概念解释
1. java解释性语言，需要编译后执行
2. java可跨平台
3. JDK=JRE(JVM+JAVASE标准类库)+开发工具集（Java编译工具）
4. JRE：Java运行环境，如果要运行开发好的.class文件，只要JRE
5. JVM；java虚拟机，不同操作系统有不同的jvm

# 二.转义字符
1. 制表符    \t
2. 一个\     \\
3. 一个"     \"
4. 一个'     \'
5. 一个回车  \r  
注意：回车回到本行
例：System.out.printIn("我的笔记/r你的")
结果：你的笔记

# 三.数据类型
## 数据类型分类
### 基本数据类型 [字节byte]
整数类型：byte[1],short[2],int[4],long[8]
浮点类型：float[4],double[8]
字符型：char[2]
布尔型：boolean[1]
### 引用数据类型
类：class
接口：interface
数组：[]

### 整型细节
